프로젝트명: Sporty-Us
한 줄 소개: 사용자 맞춤 스포츠 동호회 탐색 및 실시간 양방향 매칭 플랫폼

1. 기획 배경 및 목표

'Sporty-Us'는 파편화되어 있는 아마추어 스포츠 팀 정보를 체계적인 디렉토리로 통합하여, 사용자가 자신에게 딱 맞는 동호회를 한눈에 찾고 즉시 가입할 수 있도록 돕는 웹 기반 매칭 포털입니다. 평소 아이스하키 등 다양한 아마추어 스포츠 활동에 직접 참여하며 느꼈던 '지역 기반 팀 정보 접근의 한계'와 '폐쇄적인 가입 절차'라는 실질적인 문제를 해결하고자 기획했습니다.

2. 핵심 기능

  i) 로그인 시스템
  사용자 진입 장벽을 최소화하기 위해 별도의 복잡한 회원가입 절차를 과감히 생략했습니다. 첫 로그인 시 입력한 정보로 계정이 자동 생성되며, 이후 비밀번호 기반으로 검증하는 일원화된 인증 흐름을 구축했습니다.

  ii) 다크 테마 기반의 직관적 UI와 다중 조건 필터링
  스포티함을 살린 네온,다크 테마 UI를 적용하여 시각적 몰입감을 높였습니다. 구기 종목, 라켓 종목에서 이색 스포츠까지 폭넓은 카테고리를 제공하며, 지역별 필터와 키워드 검색을 통해 직관적인 팀 탐색이 가능합니다.

  iii) 데이터 무결성과 평가 편의성의 조화
  실제 서비스 런칭을 목표로 무단 크롤링 데이터를 배제하고, 100% 사용자가 직접 등록하는 UGC 방식을 채택했습니다. 동시에 프로젝트 원활한 시연과 기능 평가를 위해, 간편하게 켜고 끌 수 있는 '데모 데이터 토글' 환경을 구축하여 필터링 및 매칭 알고리즘을 즉시 체험할 수 있도록 설계했습니다.

  iv) 가중치 기반 맞춤형 큐레이션 알고리즘
  사용자가 설정한 선호 조건(관심 종목, 지역, 실력 수준)을 바탕으로 일치 항목당 가중치 점수를 부여하는 로직을 적용했습니다. 이를 통해 사용자에게 최적화된 동호회 리스트를 대시보드 최상단에 자동으로 큐레이션 해줍니다.

  v) 실시간 양방향 소통 및 오프라인 연결 파이프라인
  지원자는 '내 모임' 탭에서 신청 및 문의 상태를 직관적으로 트래킹할 수 있고, 팀 운영자는 전용 관리 대시보드에서 지원자를 승인하거나 문의에 직접 답변할 수 있습니다. 특히 가입이 승인되는 즉시 UI가 전환되며 팀의 카카오톡 오픈채팅 링크나 대표 연락처가 활성화되어, 온라인에서의 매칭이 오프라인 활동으로 끊김 없이 이어지도록 UX를 기획했습니다.

  vi) Firebase 기반의 실시간 데이터 동기화 
  Firebase Firestore를 연동하여 동호회 생성, 지원서 접수, 운영자의 승인 상태 변경 및 1:1 문의 등 모든 데이터 트랜잭션이 페이지 새로고침 없이 사용자 화면에 실시간으로 반영되도록 구현했습니다.
