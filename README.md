'Sporty-Us'는 사용자에게 딱 맞는 아마추어 스포츠 동호회를 한눈에 찾고 즉시 가입 신청을 할 수 있도록 돕는 웹 기반 매칭 포털입니다. 평소 아이스하키를 비롯한 여러 아마추어 스포츠 활동에 참여하면서, 흩어져 있는 지역 기반의 팀 정보들을 체계화된 디렉토리로 모아보고 싶다는 실질적인 필요성을 느껴 제작하게 되었습니다. 

1. 로그인: 사용자 편의를 위해 별도의 회원가입 페이지 없이, 첫 로그인 시 자동으로 계정이 생성되고 이후부터는 비밀번호로 검증하는 통합 인증 흐름을 구현했습니다. 비밀번호는 수정 가능하도록 설정하였습니다. 

2. 종목 및 지역 필터링: 축구, 야구 같은 구기 종목부터 테니스, 배드민턴 등의 라켓 종목, 그리고 아이스하키와 서핑 같은 이색 스포츠까지 폭넓은 카테고리를 제공합니다. 사용자는 지역별 필터 버튼과 텍스트 검색 기능을 통해 원하는 조건의 팀을 찾을 수 있습니다.

3. 실제 서비스 런칭을 염두에 두고 데이터 무단 도용을 배제한 100% 사용자 직접 등록 방식을 채택했습니다.
단, 원활한 기능 평가를 위해 토글 버튼으로 간편하게 제어할 수 있는 '테스트용 샘플 데이터' 환경을 별도로 구현하여 매칭 및 필터링 알고리즘을 직접 체험해 보실 수 있도록 설계했습니다.

4. 양방향 가입 신청 및 권한 관리: 사용자는 마음에 드는 팀에 즉시 가입을 신청 및 문의할 수 있으며, '내 모임' 탭에서 자신의 신청 상태(승인 대기 중 등)를 파악할 수 있습니다. 반대로 팀 운영자는 별도의 '신청관리, 문의관리' 페이지를 통해 지원자 목록 및 문의사항을 확인하고 가입을 직접 승인하거나 문의에 답하는 양방향 매칭 구조를 구현했습니다.

5. 사용자 맞춤형 추천 시스템: 사용자가 자신이 선호하는 종목, 지역, 실력 수준(초급~상급)을 모달 창에서 설정하면, 해당 조건에 맞춰 점수를 부여하여 가장 적합한 동호회를 최상단에 추천해 주는 로직을 적용했습니다.

6. Firebase 기반 실시간 데이터 연동: Firebase Firestore를 데이터베이스로 연동하여, 새로운 동호회 생성, 지원서 접수, 운영자의 승인 상태 변경 등의 모든 데이터 트랜잭션이 시스템에 실시간으로 반영되도록 설계했습니다.
