Sporty-Us는 일상과 스포츠를 가장 쉽게 연결하는 하이퍼로컬 취미 운동 매칭 플랫폼 입니다.
분야별, 종목별로 관심있는 스포츠에 관련된 동호회 정보를 제공합니다. 

Sporty-Us를 구성하는 3가지 핵심 기둥은 다음과 같습니다:
1. 연결성 
스포츠의 즐거움을 느낄 수 있도록 돕습니다. 구기 종목부터 아이스하키 같은 이색 종목까지, 파편화되어 있던 동호회 정보를 한곳에 모아 진입 장벽을 혁신적으로 낮췄습니다.

2. 데이터 기반 매칭
실제 운영 중인 팀의 실력대, 활동 지역, 연령대를 정교하게 필터링합니다. "가서 민폐 끼치면 어떡하지?"라는 걱정을 "내 수준에 딱 맞는 팀이네!"라는 확신으로 바꿔줍니다.

3. 스포티 라이프스타일 
단순한 팀 찾기 기능을 넘어, 사용자의 일상이 더 활기차고 역동적으로 변하도록 유도합니다. "당신의 일상이 더 스포티해지는 가장 쉬운 방법"이라는 슬로건처럼, 운동을 특별한 이벤트가 아닌 자연스러운 삶의 조각으로 만드는 것이 이 사이트의 궁극적인 목적입니다.
