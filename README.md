Sporty-Us는 일상과 스포츠를 가장 쉽게 연결하는 하이퍼로컬 취미 운동 매칭 플랫폼 입니다.
분야별, 종목별로 관심있는 스포츠에 관련된 동호회 정보를 제공합니다. 

이름과 전화번호를 입력하고 시작하기를 누르고 종목을 선택하면 조건에 맞는 팀들의 리스트가 제공됩니다. 
팀을 클릭하면 상세정보와 '신청하기'가 나타나는데, 신청하기를 누르면 이름과 전화번호가 firebase database로 전달됩니다. 
https://console.firebase.google.com/u/0/project/sporty-us/firestore/databases/-default-/data/~2Fapplications~2Fz47gtcHAv4mBPROBsYke
하단의 '내 모임'에서 신청여부를 확인하고 취소할 수도 있습니다. 

아래 사이트들에서 스포츠 별 팀들의 정보를 가지고 엑셀로 정리한 뒤 업데이트 했습니다.
